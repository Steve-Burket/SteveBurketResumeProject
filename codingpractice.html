HERE WE GO


Hi, my name is Steve Burket. This is my first HTML document and is a way for me to practice everything that I've been learning to build websites. I don't know exactly when I became interested in becoming a coder. It kind of feels like I've always been interested in technology but was afraid to take the plunge until recently. I've always found computers interesting since I was a kid. I remember dial up modems and hearing my parents' computer connect to the internet in the mid-90's bringing me into a world of new possibilities. Things have come quite a long way since then and now I'm trying to catch up as quick as I can! I feel like this is still just the beginning of something very exciting and new so I want to be a part of it. I hope to learn as much as I can about coding and put the knowledge to good use to help the world become more connected and effectively communicate with each other. There is great power that lies between the circuit boards and silicon inside these computers so with great power of course comes great responsibility which is why I try to not lose sight of the end goal which is to help humanity become better as well as be conscious that this is only a tool to that end. Fire was an amazing discovery that humans made however long ago and can be used for good as well as bad. So too is the balance with technology. It can be used to create warmth and light but if used ignorantly can yield disastrous outcomes. So, it is very important to remember why I am getting into this which is to create beauty and communicate logic to elevate our universal consciousness. I'm not alone in the endeavor as it takes many people to build greatness; luckily the world is more connected now than ever so I just need to know where to look.


I started by picking up a book from the library called "Coding For Dummies". When I first started learning guitar I read from the "Guitar For Dummies" book and it was a good start so I figured the same would be for coding which it was. I started with HTML, then started learning a little about CSS, but what I'm really excited for and can't wait to dig into more is Java Script! I want to build interactive websites that gives the user a fun experience! Slowly but surely I would also like to get into Python and as many other languages as I can. As a matter of fact I want to learn everything about computers from hardware to software and beyond. So, I have a long ways to go! Let's keep going!
